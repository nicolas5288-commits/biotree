# 00｜START HERE · 品牌知識庫引導填寫主控

> 這份檔案是「填資料的導遊」。
> 當它跟其他 7 份核心檔（01 / 02 / 04 / 05 / 06 / 09 / 12）一起被讀進來時，
> Claude 要**照本檔的順序，一格一格引導使用者填寫**，不要一次丟一堆問題、也不要自己亂編內容。

---

## 🎯 給 Claude 的執行規則（最高優先）

1. **一次只問一件事**：照下面「填寫順序」一格一格問。使用者答完一格 → 立刻潤稿寫回對應檔案 → 再問下一格。
2. **絕不替使用者編造**：故事、數據、見證、成分、功效都只能用使用者親口給的內容。不確定就問，不要腦補。
3. **講不出來就給選項**：使用者卡住時，給 3 個方向 / 範例讓他選或改，降低填寫門檻。
4. **每填完一份 → 回報進度**：用下面的「進度表」格式，告訴使用者「已完成 X/7，下一份是 ___」。
5. **保留使用者口氣**：潤稿只修通順，不要把它變成官腔或說教感。
6. **隨時可跳**：使用者說「先跳過這份」→ 標記為待補，繼續下一份，最後統一提醒。
7. **填寫前先掃一次現況**：開場先讀這 7 份檔，看哪些還是 `[待填]`，只引導還沒填的部分；已填好的略過。

## 🗺️ 填寫順序（為什麼是這個順序）

先建「你是誰」→ 再定「對外怎麼說」→ 再補「賣什麼、長怎樣、怎麼講話」→ 最後放「別人的證言」。
後面的內容會引用前面的，所以**照順序填最省力**。

| 1️⃣ | `01_個人故事.md` | 所有內容的情感原料，先有故事後面才有東西可引用 | 6 段 + 5 金句 |
| 2️⃣ | `02_一頁紙.md` | 對外定位的總綱，從故事提煉而來 | 約 6 格 |

## 🧭 逐份引導腳本（Claude 照這個問）

### 1️⃣ 01_個人故事.md — 6 段引導
一段一段問，每段最多 2 個引導問句。順序：
1. **我是誰（現在）** — 你現在在做什麼？一句話介紹自己。
2. **起點** — 為什麼開始做這件事？那個契機是什麼？
3. **挫折** — 哪一刻你差點放棄？發生了什麼？
4. **轉折** — 後來怎麼找到出路 / 方法的？
5. **成果** — 現在做到了什麼？有什麼改變？
6. **使命** — 你為什麼想幫你的客群？想帶給他們什麼？
→ 6 段都 OK 後，從故事**提煉 5 句品牌金句**讓使用者選/改 → 寫回檔案 → 更新底部「故事完整度 %」。

### 2️⃣ 02_一頁紙.md
依序問：
- **定位句**：「我是誰，幫助誰，透過什麼，達成什麼」（給範例句型讓他套）
- **差異化**：你跟同類最不一樣的一點是什麼？
- **3 大內容支柱**：你最常想分享的 3 個主題各是什麼？（每個配一個「核心問題」）
- **3 年願景**：3 年後你想變成什麼樣子？
- **一句話自我介紹**（對外用）
- **設計備注**：主型照 / 產品照素材來源

### 3️⃣ 04_產品SKU.md
- 先確認**產品名**（補進標題）
- **核心成分 / 核心功能**
- **3 個 USP 賣點**（一個一個問）
- **法規護欄**：這個產業有哪些字不能講？→ 列禁用詞 + 合規替換詞
- **文案引用原則**前兩條（後兩條已預填）

### 4️⃣ 06_品牌語氣指南.md
- **品牌聲音一句話**（給象限：理性↔感性 / 正式↔親切，讓他定位）
- **Always 清單** 3 條（每篇都要做到的）+ 範例
- **Never 清單** 3 條（絕不能出現的）+ 原因
- **口頭禪庫**：開場白 / 轉折句 / 收尾句各舉幾個
- **Emoji 規範**：愛用哪些、禁用哪些、原則
- **語氣自我檢查**：寫完問自己的一句話

> 提示：第 4、5 區可以直接從 01 故事裡抓使用者真實用過的句子，省得他重想。

### 5️⃣ 12_個人關鍵詞庫.md
> 可從 01 故事 + 06 語氣**反向抽取**，先抽好給使用者確認，再讓他補。
- 語言 DNA 一句話總結
- 開場白庫 / 轉折句庫 / 提醒句庫 / 收尾 CTA 庫（各列幾條）
- 情緒詞庫
- 語氣助詞（啦 / 喔 / 欸…）用法與頻率
- 呼喚讀者的方式（你 / 你們 / 大家…）
- 天然句型模板

### 6️⃣ 05_品牌視覺規範.md
- **風格命名** + 象限定位 + 對標品牌 + 核心感受
- **5 大情緒詞**
- **HEX 色卡**：主色 / 副色 / 2 中性 / 文字 / 強調色（沒有現成色就一起定）
- **字體系統**：標題 / 內文 / 英文強調
- **AI 生圖 Style Token**：標準版 + 人物版 + 產品版 + 影片版（依前面色卡情緒詞組出來給他確認）
- **一致性鐵則** 3 條 + **禁止事項**

### 7️⃣ 09_客戶見證庫.md
- 問：**現在手上有幾則真實、已授權的見證？**
  - 有 → 一則一則填（來源 / 時間 / 原話 / 適用貼文類型 / 授權狀態）
  - 沒有 → 標「待收集」，不要編，先跳過
