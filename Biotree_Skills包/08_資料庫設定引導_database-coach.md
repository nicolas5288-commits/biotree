# Claude Database Coach · 資料庫設定引導 ⭐

> 📂 讀取並執行 Skill：`claude-database-coach`

## 為什麼這個 Skill 必做

**3 個學員 setup 卡關**：
1. 「Project Knowledge 跟 Custom Instructions 差在哪？」
2. 「12 份檔案到底要先做哪份？」
3. 「Claude 怎麼知道我是誰？」

**做完這個 Skill** → 學員會有：
- ✅ 1 個 Claude Project 已建好（命名 + 結構）
- ✅ 50 個 Skills 已裝進 Project Knowledge
- ✅ 12 份個人品牌檔案已上傳
- ✅ Custom Instructions 已設定（讓 Claude 永遠記得你）
- ✅ 通過驗證測試（Claude 寫的貼文「像你」）

## 工作流程（總共約 60-90 分鐘）

### 階段 1 · 建立 Claude Project（10 分鐘）

#### Step 1.1 · 建立新 Project
```
1. 打開 claude.ai
2. 左側欄找 "+ New Project" → 點下去
3. 命名：「[你的名字] × AI 內容」
   範例：「林雅婷 × AI 內容」
4. 描述（選填）：「個人品牌 + 代理內容生成」
5. 點 "Create"
```

#### Step 1.2 · Project 結構檢查
進去 Project 後應該看到 3 個區塊：
- **Project Knowledge**（左側）：放 Skills + 12 份檔案
- **Custom Instructions**（左側設定）：放永久指令
- **對話區**（中間）：跟 Claude 對話

⚠️ 如果看不到 Project Knowledge → 確認你訂閱的是 **Pro** 不是免費版。

---

### 階段 2 · 安裝 50 個 Skills（10 分鐘）

#### Step 2.1 · 解壓資源包
```
1. 找到老師給的「課前準備資源包.zip」
2. 雙擊解壓縮
3. 找到 "01_Claude_Skills" 資料夾（裡面有 12 個分類資料夾）
```

#### Step 2.2 · 拖進 Project Knowledge
```
1. 回到 Claude Project
2. 點左側 Project Knowledge → "Upload files"
3. 把整個 "01_Claude_Skills" 資料夾<strong>拖進去</strong>
4. Claude 自動展開所有 SKILL.md（共 50 個）
5. 上傳完顯示「✅ 50 files added」
```

#### Step 2.3 · 驗證 Skills 已裝
在對話框試輸入：
```
你能看到我裝了哪些 Skill？列出前 10 個。
```
Claude 應該回 10 個 Skill 名稱。如果回「我看不到任何檔案」 → 重新拖檔。

---

### 階段 3 · 12 份個人品牌檔案上傳引導（30-60 分鐘）

按順序檢查 12 份檔案，每份缺 → 引導學員啟動對應 Skill 產出。

| # | 檔案名 | 對應 Skill | 狀態 |
|---|---|---|---|
| 01 | `01_品牌一頁紙.md` | `brand-foundation-builder` | ⬜/✅ |
| 02 | `02_個人品牌核心故事.md` | `six-section-story-writer` | ⬜/✅ |
| 03 | `03_目標客群輪廓.md` | `customer-persona-builder` | ⬜/✅ |
| 04 | `04_產品SKU資料.md` | `product-sku-formatter` | ⬜/✅ |
| 05 | `05_品牌視覺規範.md` | `visual-style-explorer` | ⬜/✅ |
| 06 | `06_品牌語氣指南.md` | `brand-voice-curator` | ⬜/✅ |
| 07 | `07_過去產出範本.md`（5 則最像你的貼文）| 學員自己整理 | ⬜/✅ |
| 08 | `08_社群數據基線.md`（粉絲數 / 互動率）| 學員自己統計 | ⬜/✅ |
| 09 | `09_客戶見證庫.md`（5-10 則授權見證）| 學員自己蒐集 | ⬜/✅ |
| 10 | `10_常見Q&A.md`（10 題客戶常問）| 學員自己列 | ⬜/✅ |
| 11 | `11_30天行動方案.md` | `30day-coach`（課後產出）| ⬜/✅ |
| 12 | `12_個人關鍵詞庫.md`（口頭禪 / 開頭 / 結尾）| 從 06 自動提煉 | ⬜/✅ |

**Claude 自動執行**：
```
1. 詢問學員「12 份你已經有哪幾份？」
2. 列出缺的，依優先序引導：01 → 02 → 03 → 06 → 04 → 05 → 其他
3. 每份缺 → 詢問「現在要啟動 [對應 Skill] 嗎？」
4. 學員 yes → 啟動該 Skill → 完成後回到 database-coach
5. 12 份全 ✅ 才進階段 4
```

⚠️ **最低必備 6 份**（沒做完不要進階段 4）：
- 01 一頁紙、02 故事、03 客群、04 SKU、06 語氣、12 關鍵詞庫

---

### 階段 4 · Custom Instructions 設定（5 分鐘）

#### Step 4.1 · 開啟 Custom Instructions
```
1. Claude Project → 設定 → Custom Instructions
2. 看到一個輸入框（可以貼 1500 字）
```

#### Step 4.2 · 貼上 永久指令模板

```
你是我的 AI 內容創作助理。

我的個人品牌資料在 Project Knowledge 12 份檔案：
- 01 我的品牌一頁紙
- 02 我的個人故事
- 03 我的目標客群輪廓
- 04 產品 SKU 資料
- 05 我的品牌視覺規範
- 06 我的品牌語氣指南
- 07 我過去最成功的 5 則貼文
- 08 我的社群數據基線
- 09 我的客戶見證庫
- 10 客戶常問 Q&A
- 11 我的 30 天行動方案
- 12 我的個人關鍵詞庫

每次我請你寫文案 / 生圖 / 拍影片，你都要：
1. 先看 06 語氣指南的 Always / Never
2. 對著 03 客群（主力 persona）寫
3. 用 02 故事段落 + 12 關鍵詞庫
4. 04 SKU 資料是產品功效的唯一依據（不要憑空編）
5. 法規禁字：治療 / 治癒 / 100% / 神奇 / 醫師推薦 一律不用

我的素材在 Higgsfield Media（透過 MCP show_medias 看得到）：
- M01 主型照（我的正面照）
- M05 真實產品照
- 其他 M02-M60 是 AI 自動生的多場景照

所有生圖 / 生影片需求都用 MCP 直接執行，
絕對不要叫我去 Higgsfield 網站操作。

回覆風格：
- 簡短直接、不囉嗦
- 中文為主，避免英文夾雜
- 每次給結論在前、解釋在後
- 寫文案時直接給 3 個版本（短 / 標準 / 長）讓我選
```

#### Step 4.3 · 存檔
點 "Save Custom Instructions" → 看到「✅ Updated」。

---

### 階段 5 · 測試驗證（10 分鐘）

#### 測試 1 · Claude 知道你是誰嗎？
```
輸入：「先載入品牌包，然後告訴我你看到我的定位句是什麼？」

正確回答：「✅ Brand Pack 已載入。你的定位句：『[從 01 讀出來的句子]』」

錯誤回答：「我不知道你的定位句是什麼」 → 重新檢查 01 檔案是否上傳成功
```

#### 測試 2 · Claude 寫的「像你」嗎？
```
輸入：「啟動 post-writer-12-types Skill，類型：個人故事。
       主題：我下午 3 點頭痛但不能放下小孩。寫標準版。」

檢查：
✅ 開頭有具體場景（不是「你是不是」）
✅ 第一人稱
✅ 有具體日期 / 數字
✅ 有口頭禪 / 自嘲
✅ 結尾不是 CTA
✅ 沒禁字

如果有 1-2 條不符 → 重新確認 06 語氣指南是否上傳完整。
```

#### 測試 3 · Claude 知道你的客群嗎？
```
輸入：「對 Tina（我的 persona 1）寫一個下午頭痛的限動文案」

正確回答：用 03 客群輪廓的 Tina 痛點（下午 3 點頭痛）+ 對應的金錢觀 / 媒體習慣寫
錯誤回答：寫一個泛泛的「都會女性」貼文 → 重新檢查 03
```

3 個測試都過 → ✅ 資料庫設定完成。

## Bug 排查

| 症狀 | 原因 | 解法 |
|---|---|---|
| 看不到 Project Knowledge | 不是 Pro 版 | 升級 Pro |
| Skills 上傳失敗 | 檔案 > 100 個 | 分批拖（每次 30-50 個） |
| Claude 說「我看不到檔案」 | 檔案沒同步完 | 等 30 秒 + 重新整理頁面 |
| Claude 寫的不像你 | 06 語氣指南沒寫好 | 重做 voice-curator + 加細節 |
| Custom Instructions 沒生效 | 對話沒重開 | 開新對話再試 |
