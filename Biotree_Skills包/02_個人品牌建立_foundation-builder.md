# Brand Foundation Builder · 個人品牌基礎建立 ⭐

> 📂 讀取並執行 Skill：`brand-foundation-builder`

## 為什麼這個 Skill 必做

**3 個學員常見問題**：
1. 「我不知道我能教什麼」
2. 「我跟一堆做保健品代理的人有什麼不一樣？」
3. 「我寫貼文沒方向、沒主軸、東一篇西一篇」

**根因**：沒有「定位句」。沒有定位句就像沒地址 — 客戶找不到你，你也不知道自己在哪。

**做完這個 Skill** → 學員會有：
- ✅ 1 句定位句（記得住、講得出）
- ✅ 1 頁紙（隨時看自己 = 不迷失）
- ✅ 5 個內容支柱（後續貼文 / Reels 都從這 5 個方向發想）
- ✅ 1 年 / 5 年願景（避免短視）

## 工作流程（總共約 30 分鐘）

### 階段 1 · 10 探索問題（10 分鐘）

**鐵則**：每題學員<strong>口語回答</strong>（語音 / 打字都可），Claude 整理。每題給 1-3 分鐘思考。

#### Q1 · 過去 3 年做最多次的事
**問**：不是「最自豪」也不是「最賺」 — 是<strong>「次數最多」</strong>的事。
**範例答**：「跟客戶在 LINE 一對一聊產品、聊她們的生活」

#### Q2 · 做什麼會忘記時間
**問**：哪件事你做著做著抬頭看時間發現過 3 小時？
**範例答**：「翻 IG 看別人怎麼設計貼文 / 跟客戶 LINE 聊她的小孩」

#### Q3 · 別人最常稱讚你的點（不是你以為的）
**問**：朋友 / 客戶 / 家人最常說「妳真的很 X」 — 那個 X 是什麼？
**範例答**：「妳很會聽人講話 / 妳真的很真實 / 妳推薦的東西我都用得上」

#### Q4 · 走過最遠的彎路
**問**：3 年內你做過最後悔的決定是什麼？
**範例答**：「跟錯團隊 6 個月，月薪掉 90%」

#### Q5 · 18 歲後 3 個跟別人不一樣的選擇
**問**：你做過哪些「同年人沒做的」決定？
**範例答**：
- 27 歲離開穩定工作做代理
- 30 歲生第二胎堅持親餵 18 個月
- 33 歲拒絕加入大團隊改自己做

#### Q6 · 只能教 1 件事會教什麼
**問**：如果你只能教世界上 1 件事（賺錢 / 育兒 / 任何）你會教什麼？
**範例答**：「教媽媽怎麼在帶小孩的同時，還能找到一個只屬於自己的 10 秒儀式」

#### Q7 · 最常被諮詢的領域
**問**：朋友 / 客戶 LINE 你問問題，最常問什麼？
**範例答**：「保健品怎麼挑」「累爆怎麼辦」「怎麼跟老公分擔家務」

#### Q8 · 最自豪的成就
**問**：不是「我月薪 X 萬」 — 是<strong>內在自豪</strong>的事。
**範例答**：「我有 3 個跟我認識超過 5 年的客戶，她們會在感冒時 LINE 我而不是醫師」

#### Q9 · 最不能忍受的事
**問**：什麼事看到你會直接退追蹤 / 翻白眼？
**範例答**：「KOL 假裝完美 / 用『家人們』開頭的網紅腔 / 說『XX 神奇治百病』」

#### Q10 · 100 萬粉絲想跟他們講什麼
**問**：如果明天 100 萬人在等你發言 30 秒，你會講什麼？
**範例答**：「妳不需要變成更厲害的人才能被愛。妳只需要先看見自己」

---

### 階段 2 · 3 圈交集（5 分鐘）

把 10 題答案整理成 3 個圈，找<strong>交集</strong>。

#### 圈 1 · 你會的（3 件，從 Q1 / Q3 / Q7 提煉）
```
1. 跟客戶建立 5+ 年信任關係
2. 用 LINE 一對一聊出客戶真痛點
3. 把產品故事跟自己生活綁在一起寫貼文
```

#### 圈 2 · 市場要的（客群核心需求 — 從 03 客群輪廓抓）
```
1. 30-45 歲媽媽要「快速、有效、不貴」的保健方法
2. 想被「真實的人」推薦，不是被網紅推銷
3. 找一個能諮詢的「對的人」，不只買產品
```

#### 圈 3 · 你愛的（3 件，從 Q2 / Q6 提煉）
```
1. 跟人聊她們的真實生活
2. 把「累、煩、迷失」變成可以被聽見的故事
3. 慢慢來、不催促
```

#### 交集 = 你的「不可替代位置」

```
你會的 + 市場要的 + 你愛的
        ↓
「30-45 歲女人想被真實的人在 LINE 一對一聽見痛點 + 用保健品慢慢恢復」
```

---

### 階段 3 · 定位句生成（5 分鐘）

**定位句模板**：
```
我是 [優勢標籤]，
我幫 [客群描述]
解決 [核心痛點]，
用 [方法]。
```

**範例（接續上面）**：
```
我是「30-45 歲媽媽的健康陪伴者」（優勢標籤）
我幫「全職媽媽 + 中小企業上班族」（客群）
解決「下午頭痛 + 晚上累爆 + 對自己沒感」（核心痛點）
用「[產品] 使用儀式 + LINE 1 對 1 陪伴」（方法）。
```

**規則**：
- 整句不超過 80 字
- 必須能 30 秒講出來
- 必須包含具體年齡 / 痛點 / 方法（不要抽象）

---

### 階段 4 · 一頁紙（10 分鐘）

把所有資訊整合成 `01_品牌一頁紙.md`：

```markdown
# [學員名] 的品牌一頁紙 · v1.0

## 定位句（30 秒能講完）
[1 句不超過 80 字]

## 30 秒故事版（從 02_個人故事 提煉）
[100-150 字濃縮版：起點 → 轉變 → 我現在做什麼]

## 3 條核心信念
1. [例：真話比業績重要]
2. [例：慢慢來剛剛好]
3. [例：身體會講話只要妳願意聽]

## 客群描述（從 03_客群輪廓 提煉）
- 主力：[Persona 1 簡述]
- 次要：[Persona 2 簡述，如有]

## 我能提供
1. 產品：[你的品牌] [3 支主打 SKU]
2. 服務：LINE 1 對 1 諮詢 / IG 內容陪伴
3. 經驗：[具體年數 + 客戶數]

## 5 個內容支柱（後續所有貼文都從這 5 個方向）
1. **個人故事**（30%）— 我的低谷、轉變、現在
2. **客戶見證**（20%）— 真實客戶的轉變
3. **產品教育**（15%）— 成分、用法、為什麼選 [你的品牌]
4. **媽媽日常**（20%）— 帶小孩、自我照顧、生活感
5. **價值觀**（15%）— 反思、信念、邀請

## 1 年願景
[具體：粉絲 X / 月營業 X / 客戶 X]

## 5 年願景
[Lifestyle 描述：每天怎麼過、想幫多少人、想建立什麼影響力]

## 優勢標籤（給後續 Skill 用）
- [標籤 1] 例：「30-45 歲媽媽的健康陪伴者」
- [標籤 2] 例：「LINE 一對一信任建立者」
- [標籤 3] 例：「真話派代理」
```

回報「✅ 已同步 01_品牌一頁紙.md（[字數]）+ 優勢標籤 3 條」。

## Bug 排查

| 症狀 | 原因 | 解法 |
|---|---|---|
| 學員 10 題答不出來 | 太空泛 | 給 1-2 個範例答案讓他抄改 |
| 3 圈找不到交集 | 圈 1 / 圈 3 內容雷同 | 強迫圈 3「不能跟賺錢有關」（純興趣） |
| 定位句寫成「我是賣保健品的」 | 沒提煉優勢標籤 | 重做 Q3 + Q7（別人怎麼看你 / 最常被諮詢） |
| 5 內容支柱比例失衡 | 想全寫產品教育 | 強制「產品教育不超過 20%」，70% 必須是個人故事 + 客戶見證 + 日常 |
| 願景寫「賺很多錢」 | 物質目標不耐久 | 重問「Lifestyle 是什麼？早晨怎麼過？」 |
