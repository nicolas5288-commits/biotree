# Hook Line Generator

> 📂 讀取並執行 Skill：`hook-line-generator`

## 觸發時機
- 學員說「給我 hook」「想 5 個開頭」「Hook 句變體」

## 5 種 Hook 風格
1. 疑問句版（「你是不是也⋯」）
2. 數字版（「我家小孩這個月已經第 3 次感冒」）
3. 反向版（「不要再囤保健品了」）
4. 故事版（「上週四那天我崩潰了」）
5. 情緒版（「我看完那則訊息掉淚」）

## 工作流程
1. 確認主題 + 客群
2. 每風格產 1-2 個 hook，共 5-10 個
3. 每個 ≤ 20 字
4. 標記「適合受眾」與「預期 hook 強度」

## 規則
- 不用「家人們」「不藏私」開頭
- 必須具體（時間 / 場景 / 對話）
- 跟學員 12 關鍵詞庫一致
