# CapCut Edit SOP

> 📂 讀取並執行 Skill：`capcut-edit-sop`

## 觸發時機
- 學員說「CapCut 怎麼剪」「拼接 segments」

## 標準剪輯 SOP

### Step 1：匯入順序
依 S1→Sn 排列

### Step 2：轉場
- 一般：0.3s 淡入淡出
- 對話 / 諷刺：smash cut（無轉場）
- 時間流逝：fade through black 0.5s

### Step 3：字幕
- CapCut 自動字幕
- 字體 Noto Sans TC Bold 90-110pt
- 顏色：白底 + 黑邊
- 位置：下方 1/4

### Step 4：配樂
- 音量 -20dB
- 類型依影片風格（lo-fi / vlog / cinematic）
- BGM 庫：Epidemic Sound / Pixabay / YouTube Audio Library

### Step 5：字卡
- Hook 段：「家人們 ⚠️ 必看」
- CTA 段：大字「+1」「私我」
- 進度標：「Step 1」「Day 5/30」

### Step 6：輸出
- 9:16 1080×1920 30fps
- 8-12 Mbps

## 工作流程
1. 收影片類型 + 段數
2. 套對應 SOP
3. 給逐步剪輯指示
