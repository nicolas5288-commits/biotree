# Post Writer 12 Types

> 📂 讀取並執行 Skill：`post-writer-12-types`

## 觸發時機
- 學員說「寫 [類型] 貼文」「啟動 post-writer」

## 12 種類型對應結構
1. 個人故事型 — 起點→低谷→轉變→現在
2. 低谷挫折型 — 80% 純故事 / 20% 連結
3. Before & After — Before / 中間發生 / After / 領悟
4. 客戶見證型 — 客戶背景 / 痛點 / 用多久 / 結果 / 觀察
5. 賺錢收入型 — 數字 / 過程 / 最痛 / 給想做的人建議
6. 招募型 — 6 個月前的我 / 轉捩點 / 想找誰 / CTA
7. 產品教育型 — 問題 / 一般解法 / 我們解法 / 成分證據
8. 日常融入型 — 場景 / 困擾 / 自然帶產品 / 感想
9. 痛點共鳴型 — 「你是不是也」+ 5-7 痛點 + 「中幾個」
10. 反向觀點型 — 大家說 X / 我覺得 Y / 3 條理由
11. FAQ 型 — 問題 / 簡答 / 詳細 / 適合誰
12. 價值觀宣言型 — 我相信 X / 為什麼 / 邀請同頻
