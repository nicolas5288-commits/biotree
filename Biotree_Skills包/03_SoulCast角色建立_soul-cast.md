# Soul Cast Character Builder

> 📂 讀取並執行 Skill：`soul-cast-character-builder`

## 觸發時機
- 學員說「建 Soul Cast」「角色一致性」

## 工作流程

### Step 1：準備素材
- 1 張代理人正面照（1080p+ 自然光）
- 已克隆好的聲音 ID（從 voice-clone-script-writer）

### Step 2：在 Higgsfield Soul Cast 建立
1. 上傳正面照
2. 綁定克隆聲音
3. 設定基本資料：
   - 角色名稱（建議用「[姓名]_主角色」）
   - 風格描述（從 03 客群輪廓擷取）

### Step 3：測試
用一段短腳本測試：
- 臉看起來像不像本人
- 聲音對嘴自然不
- 風格一致不

### Step 4：跨影片重用
- 複製 Soul Cast ID
- 後續每支 Reels / 廣告都引用該 ID
- 每段 Higgsfield Prompt 的 Character token 用該 ID

### Step 5：建立第二個 Soul Cast（可選）
- 為對話腳本 Skit 建另一個造型版本
- 同人不同服裝 / 髮型

## 規則
- 每帳號可存多個 Soul Cast
- 跨影片角色一致性靠這個鎖定

## 載入回覆
"✅ Soul Cast Builder 啟動。你已克隆好聲音了嗎？正面照準備好了嗎？"
