# Voice Clone Script Writer

> 📂 讀取並執行 Skill：`voice-clone-script-writer`

## 觸發時機
- 學員說「2 分鐘錄音腳本」「Higgsfield 聲音克隆要念什麼」

## 標準腳本結構（2 分鐘 = 4 段 × 30 秒）

### Part 1：日常自我介紹（30s）
要素：自我介紹 + 角色 + 過去 3 年轉變

### Part 2：情緒變化句（30s）
要素：你知道嗎 / 我以前 / 直到那一天 + 一個轉折

### Part 3：數字與停頓（30s）
要素：30 天 / 5 個 / 3 個 + 刻意停頓「⋯」

### Part 4：品牌字眼 + CTA（30s）
要素：你的品牌 / 產品名 / 我自己也在用 / 留言、私訊

## 客製化來源
- 從 02 一頁紙擷取「我是誰」
- 從 04 SKU 擷取主推產品
- 從 12 關鍵詞庫擷取口頭禪
- 從 03 客群擷取共鳴點

## 念稿 5 重點
1. 平常講話速度
2. 情緒真的進去
3. 數字清楚
4. 品牌字眼念清楚
5. CTA 溫暖

## 載入回覆
"✅ Voice Clone Script Writer 啟動。你的名字 / 主推產品是？"
