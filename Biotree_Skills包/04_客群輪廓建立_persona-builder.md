# Customer Persona Builder · 客群輪廓建立 ⭐

> 📂 讀取並執行 Skill：`customer-persona-builder`

## 為什麼這個 Skill 很重要

**寫貼文最大的錯：「對所有人寫 = 沒人有感」**。

學員常見問題：
- 「我的客戶是 25-50 歲女性」← 太廣，包山包海，根本沒法寫
- 「我的客戶就是想養生的人」← 抽象到沒有畫面
- 寫的貼文「都對」但「都不痛」 = 客戶滑過不停留

**正確做法**：選 1-2 個**最會買、最會分享**的「典型客戶」，給她取名、寫她的一天、列她的 5 個痛點 5 個慾望 — 之後寫貼文就「對著她寫」。寫完不滿意問自己：「Tina 看到這篇會停下來嗎？」沒停就重寫。

**1 個好的 persona = 後續 100 篇貼文都有著力點**。

## 工作流程（總共約 30-45 分鐘）

### 階段 1 · 基本輪廓（5 分鐘）

請學員填 8 項基本資訊，每項都要**具體到可以見面**：

| 項目 | 不可接受（太空泛） | 可接受（具體） |
|---|---|---|
| **暱稱** | 「主婦」 | 「小資媽咪 Tina」 |
| **年齡範圍** | 「30-50」 | 「34-38（核心 36）」 |
| **性別** | 女 | 女 |
| **居住地** | 「都市」 | 「新北中和 / 三重 / 板橋」 |
| **職業** | 「上班族」 | 「美編 / 文字工作者 / 中小企業行政」 |
| **月收入（家庭）** | 「中產」 | 「夫妻合計 NTD 8-12 萬」 |
| **婚姻 / 家庭** | 「已婚」 | 「已婚 5-8 年，2 個小孩 4-7 歲」 |
| **教育程度** | 「大學以上」 | 「私立大學畢，會看英文文章但不寫」 |

**規則**：每一項都要給「具體上限+下限」，不要單一數字。

---

### 階段 2 · 她的一天（精確到小時）（10 分鐘）

引導學員寫 **persona 從早上 6:30 到晚上 23:00 的時間軸**。每個時段問：
- 在做什麼？
- 心情狀態（焦慮/平靜/疲累/開心）？
- 是不是「滑手機」的時段（重要！這是內容投放點）

**範例（小資媽咪 Tina · 36 歲）**：

```
6:30   鬧鐘響 — 還想睡 5 分鐘 — 沒滑手機
6:45   起床 — 進廚房煮稀飯 — 心情：機械式 — 沒滑
7:00   叫小孩起床 — 心情：催促 + 微煩躁 — 沒滑
7:30   送小孩上學 — 開車 — 沒滑
8:00   到公司打卡 — 滑 IG 5 分鐘 ⭐⭐ ← 內容黃金時段
8:30   開始工作 — 沒滑
12:00  午餐外帶吃辦公桌 — 滑 IG 15 分鐘 ⭐⭐⭐ ← 最大時段
13:30  繼續工作 — 沒滑
17:00  下班接小孩 — 沒滑
17:30  回家做晚餐 — 沒滑
19:00  全家吃晚餐 — 沒滑
20:00  小孩寫功課 / 自己滑手機 — IG/FB 30 分鐘 ⭐⭐⭐ ← 第二大時段
21:30  幫小孩洗澡哄睡 — 沒滑
22:00  自己沖澡 — 沒滑
22:30  躺床滑手機 — IG 20 分鐘 ⭐⭐ ← 睡前
23:00  關燈睡 — 通常還在想「明天怎麼辦」
```

**結論輸出**：「Tina 滑 IG 黃金 4 個時段：8:00 / 12:00 / 20:00 / 22:30」 → 後續發文時程對著這 4 個時段。

---

### 階段 3 · 5 個痛點（具體場景，不抽象）（10 分鐘）

**鐵則**：每個痛點必須是「上週四下午 X 點具體發生的事」，不要「她一直很累」這種抽象描述。

**5 痛點公式**：
1. **身體痛**（具體場景：哪個身體部位、什麼時候、感受）
2. **時間痛**（具體場景：哪天哪段時間、要做什麼來不及）
3. **金錢痛**（具體：哪筆支出讓她猶豫、為什麼）
4. **關係痛**（具體：跟先生 / 婆婆 / 同事 / 小孩 哪次衝突）
5. **自我痛**（具體：哪天照鏡子的某一瞬間）

**範例（Tina）**：

```
痛 1（身體）：每天下午 3 點開始頭痛眼脹，晚上回家累到不想跟小孩說話。
              上週三差點對 5 歲女兒兇起來，事後在浴室哭。
              
痛 2（時間）：禮拜五晚上 8 點，老公叫她去買牛奶，
              她坐在沙發上想動但動不了。「為什麼是我去？」
              
痛 3（金錢）：上個月看到一個 NTD 1,800 的保養品，
              在蝦皮加入購物車 4 次最後沒結帳。
              「這 1,800 可以付小孩 1 個月安親費」。

痛 4（關係）：婆婆生日聚餐，婆婆說「現在年輕人就是抗壓力差」。
              她笑笑沒回，回家路上跟先生大吵一架。
              
痛 5（自我）：上週日下午 3 點，她在浴室鏡子前，看到自己
              眼下泛黑、嘴角下垂。「我才 36 歲，我長得像 46」。
```

**結論輸出**：這 5 個痛點 = 後續寫貼文的 5 個 Hook 來源。「上週日下午 3 點」這種具體日期就是貼文開頭。

---

### 階段 4 · 5 個慾望（10 分鐘）

不要寫「想要更好的生活」，要寫**畫面感的具體場景**。

**5 慾望公式**：
1. **5 年後的早晨**（具體畫面：她在哪裡、做什麼、什麼心情）
2. **被人看見的方式**（誰會說什麼話）
3. **小確幸**（每月 / 每週 1 件「為自己」的事）
4. **身體狀態**（醒來時 / 中午 / 晚上的感覺）
5. **錢的關係**（不是「賺更多」，是「不用擔心 X」）

**範例（Tina）**：

```
慾 1（5 年後早晨）：5 年後一個冬天早上，她慢慢沖一杯咖啡，
                    寫一頁日記，再開始一天。不再被鬧鐘追著跑。

慾 2（被人看見）：婆婆對她說「妳現在氣色比結婚時還好」，
                  同事看到她說「妳是不是去做了什麼？」

慾 3（小確幸）：每週六早上自己一個人去咖啡店，
                坐 1 小時、不被任何人 LINE 找。

慾 4（身體狀態）：醒來不想再睡 / 下午 3 點不頭痛 /
                  晚上 11 點還能跟先生聊 30 分鐘不會睡著。

慾 5（錢的關係）：不用擔心一個月多花 NTD 3,000 在自己身上會
                  影響家用。
```

---

### 階段 5 · 金錢觀（5 分鐘）

| 項目 | 問什麼 |
|---|---|
| **每月可支配** | 扣除生活必要支出後，每月「為自己」可花多少？ |
| **保健品價格甜蜜點** | NTD 800？1,200？1,500？2,000？2,500？超過多少會猶豫？ |
| **訂閱接受度** | 願意按月扣款嗎？(願意 / 不太願意 / 絕對不願意) |
| **分期接受度** | 願意分 3 期 / 6 期 / 12 期嗎？ |
| **首單心理門檻** | 第一次買 NTD 多少以下會「先試試看」？ |

**範例（Tina）**：「每月可支配 NTD 3,000 / 保健品甜蜜點 NTD 1,200-1,500 / 願意月訂閱（如果可隨時取消）/ 分 3 期可、6 期不要 / 首單 NTD 800 以下會直接試」

---

### 階段 6 · 媒體習慣（5 分鐘）

| 平台 | 比例 | 用法 |
|---|---|---|
| **IG** | 60% | 滑首頁 / 看限動 / 偶爾 reels |
| **FB** | 20% | 看媽媽社團 / 朋友動態 |
| **LINE** | 15% | 跟媽媽群組 / 老公 / 老師聊 |
| **YouTube** | 5% | 偶爾看食譜或育兒 |
| **Threads** | 0% | 沒在用 |
| **TikTok** | 0% | 不太信任 |

**結論輸出**：「主場是 IG，限動 > 貼文 > Reels」 → 學員火力 80% 投 IG，20% FB。

---

### 階段 7 · 她討厭什麼樣的內容（最關鍵）（5 分鐘）

⚠️ **這段最重要 — 比「她喜歡什麼」更能告訴你怎麼寫**。

**問 5 個「不要這樣」**：
1. 哪種帳號她會**馬上取消追蹤**？
2. 哪種貼文她**滑過不停留**？
3. 哪個字眼出現會讓她「翻白眼」？
4. 哪種 KOL 風格她**完全不信任**？
5. 哪種 CTA 出現她會**直接封鎖**？

**範例（Tina）**：

```
討厭 1：開口閉口「姊妹們」「親愛的家人們」 — 假甜膩
討厭 2：曬豪宅 / 名牌 / 名車 — 跟她生活脫節
討厭 3：「神奇 / 100% / 治好我的⋯」療效字 — 看了就退追
討厭 4：「私訊我加 LINE 立省 XX 元」CTA — 像詐騙
討厭 5：濾鏡開太重的自拍 — 不真實
討厭 6：純文字長文沒分段 — 看了累
討厭 7：英文夾雜（「我這個 routine 真的 super amazing」）— 假掰
```

**結論寫進 KB 的「Never 清單」** — 後續所有貼文 Claude 自動避開這 7 條。

## 進階：1 個 vs 2 個 persona 怎麼選？

| 學員狀況 | 建議 |
|---|---|
| 月營業 < NTD 5 萬 | 寫 1 個就好（核心客群） |
| 月營業 NTD 5-15 萬 | 寫 1 個主力 + 1 個次要 |
| 月營業 > NTD 15 萬 | 寫 2-3 個 persona，內容分層投放 |

⚠️ **絕對不要寫超過 3 個** — 越多越無聊，最後等於沒寫。

## Bug 排查

| 症狀 | 原因 | 解法 |
|---|---|---|
| 學員寫的 persona 太籠統 | 想包山包海 | 強迫她「給這個 persona 取名」+「想 1 個你認識的真實朋友當原型」 |
| 5 個痛點都很抽象 | 沒給具體日期 | 追問「上週這個痛點什麼時候出現？」「是哪一天？」 |
| 慾望寫成「我想賺 100 萬」 | 學員自己投射 | 重來，問「她（不是你）想要的早晨長什麼樣？」 |
| 「她討厭的」寫不出來 | 沒罵過 KOL | 問「你最近取消追蹤的帳號是誰？為什麼？」 |
| 寫完看不出跟自己品牌差異 | persona 太通用 | 加 1 個「她不是的人」對照（例：「Tina 不是月入 10 萬的 KOL，是普通上班族」）|
