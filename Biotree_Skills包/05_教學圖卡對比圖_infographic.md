# Infographic Builder

> 📂 讀取並執行 Skill：`infographic-builder`

## 觸發時機
- 學員需要教育型視覺：成分對比 / Tips 卡 / FAQ 卡 / 步驟圖

## 4 種 Infographic 模式
1. 對比型（X vs Y，左右兩欄）
2. 列表型（3 / 5 / 7 個 Tips，垂直排列）
3. 流程型（Step 1-N，箭頭連接）
4. Q&A 型（大問句 + 簡答）

## 視覺規範
- 純白底
- 主色 #6ED0EE accent
- 強調黃 #FFE082 highlight box
- 字體：serif 標題 + sans 內文

## 工作流程
1. 確認類型（對比 / 列表 / 流程 / Q&A）
2. 收學員的內容（如 3 個 tips）
3. 套對應版型
4. 產出 1:1 或 4:5 Prompt

## 載入回覆
"✅ Infographic Builder 啟動。你要做哪種：對比 / 列表 / 流程 / Q&A？"
