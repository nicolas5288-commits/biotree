# BioTree 個人品牌工具包

個人品牌 Content Factory 的 BioTree 版工具包，共 5 個模組：

| 資料夾 | 內容 |
|--------|------|
| `03_個人品牌建立工具_BioTree版` | 10 個探索問題、3 圈交集表、6 段式個人故事框架、痛點產品橋樑表、60 種素材清單 |
| `_shared` | 品牌規範、合規清單、Windsor 數據表規格 |
| `營運工具包` | 銷售報表 / IG 數據 / 投放品質 / 競品廣告分析、腳本生成、影片審核 |
| `bio tree品牌知識庫` | 12 份品牌資料檔 + 貼文 / 限動 / 輪播範本 |
| `Biotree_Skills包` | 50+ 個 Claude Code 工作 Skills（文案、視覺、影片、教練、文件處理） |

## 下載方式（Claude Code）

```
幫我把 https://github.com/nicolas5288-commits/biotree 這個 repo
clone 到 ~/Downloads/bio tree/，如果已存在就 pull 更新。
```
